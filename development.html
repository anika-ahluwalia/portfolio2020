Spotify Development
React, JavaScript | November 2020

Introduction

In order to become more comfortable with React, I decided to create a website that would allow a user to aggregate their favorite songs into a playlist. This webpage would also include functionality that would allow the user to filter and sort their list of songs that could be added to the playlist.

The goal of this project was to gain familiarity with React and web development, while still creating a useful application. This application draws inspiration from my favorite music streaming service, Spotify. While Spofity allows users to easily search through their liked songs by either title, artist, or genre, there is no easy way to filter all of these specifications at once. Personally, when I go to create a playlist in the software, I find myself endlessly scrolling through my list of 500+ liked songs until I stumble on the select few that fit the mood that I am attempting to achieve for this playlist.

Thus, this design attempts to remedy this shortcoming and demonstrate a concept of streamlined browsing for songs. While users retain the ability to search for their desired songs, they can filter these results in ways that may be helpful when creating their playlists. For example, users can filter songs based on mood and genre which may be helpful when creating a playlist of similar songs. They can also sort their songs based on the length and rating of the song, which could streamline their browsing as well.

Usability Principles

I began by considering the usability principles that I wanted to incorporate into my application:

Usability

    Actionable components are clearly separated (button and text enter UI).
    Responsive to resizes in the screen size.

Memorability

    Action buttons have the same format everywhere.
    All cards include the same information in similar displays and have the same information hierarchy.

Learnability

    Action buttons cause immediate visual changes that can easily be reverted.

Efficiency

    Filtering and sorting allows users to enrich their browsing experience.
